# Online Access Recovery Guide

Article ID: KB-0005
Category: account access issue
Audience: Supervisor
Last reviewed: 2026-06-03

## Purpose
Customer cannot sign in, pass MFA, reset password, or access mobile banking.

## Agent Procedure
1. Confirm the customer's identity using approved synthetic verification steps.
2. Review account status, case history, and related transactions.
3. Apply the category policy: Validate identity, unlock profile, reset MFA when appropriate, and warn about scams.
4. Document the decision, evidence, and any handoff queue.

## Customer-Safe Language
Use plain language. Acknowledge the concern. Explain the next step without promising outcomes that require operations review.

## Escalation
Escalate to Digital Support when documentation is incomplete, customer sentiment is high-risk, or the request exceeds frontline authority.
